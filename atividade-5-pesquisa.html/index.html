O que é o elemento HTML ?

O elemento HTML <b> é utilizado para representar um trecho de texto, geralmente exibido em negrito. No entanto, ele não transmite nenhum significado adicional. Sem indicar importância ou ênfase no conteúdo.

Exemplo

Este produto está com 50% de desconto apenas hoje!

Isso fará com que a frase “50% de desconto” apareça em negrito, chamando a atenção do leitor.
